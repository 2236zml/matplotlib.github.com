=======
Matshow
=======

Simple `~.axes.Axes.matshow` example.


.. code-block:: default

    import matplotlib.pyplot as plt
    import numpy as np


    def samplemat(dims):
        """Make a matrix with all zeros and increasing elements on the diagonal"""
        aa = np.zeros(dims)
        for i in range(min(dims)):
            aa[i, i] = i
        return aa


    # Display matrix
    plt.matshow(samplemat((15, 15)))

    plt.show()
