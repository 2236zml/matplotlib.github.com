matplotlib.pyplot.xscale¶

matplotlib.pyplot.xscale(value, **kwargs)[source]¶

Set the x-axis scale.

Parameters:
value : {"linear", "log", "symlog", "logit", ...}

scaling strategy to apply

See also

matplotlib.scale.LinearScale
linear transform
matplotlib.scale.LogTransform
log transform
matplotlib.scale.SymmetricalLogTransform
symlog transform
matplotlib.scale.LogisticTransform
logit transform

Notes

Different kwargs are accepted, depending on the scale. See the scale module for more information.
