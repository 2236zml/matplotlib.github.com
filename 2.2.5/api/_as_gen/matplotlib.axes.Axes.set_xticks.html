matplotlib.axes.Axes.set_xticks¶

Axes.set_xticks(ticks, minor=False)¶

Set the x ticks with list of ticks

Parameters:
ticks : list

List of x-axis tick locations.

minor : bool, optional

If False sets major ticks, if True sets minor ticks. Default is False.
