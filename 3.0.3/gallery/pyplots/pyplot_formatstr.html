Pyplot Formatstr¶

Use a format string to colorize a plot and set its markers.

import matplotlib.pyplot as plt
plt.plot([1,2,3,4], [1,4,9,16], 'ro')
plt.axis([0, 6, 0, 20])
plt.show()
